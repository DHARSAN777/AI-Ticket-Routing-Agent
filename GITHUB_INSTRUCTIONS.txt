🔐 YOU'LL NEED
═══════════════════════════════════════════════════════════════════════════

1. GitHub Account (free)
   - Sign up: https://github.com
   - Already signed up: Skip this

2. Git Installed
   - Download: https://git-scm.com/download/win
   - Or already installed: Skip this

3. Authentication (Choose one)

   Option A: Personal Access Token (RECOMMENDED)
     1. Go: https://github.com/settings/tokens
     2. Create token with "repo" scope
     3. Copy token
     4. Use as password when pushing
   
   Option B: GitHub CLI
     1. Download: https://cli.github.com/
     2. Run: gh auth login
     3. Auto-authenticated after setup

   Option C: SSH Key (Advanced)
     1. Generate key: ssh-keygen -t rsa -b 4096
     2. Add to GitHub: https://github.com/settings/keys
