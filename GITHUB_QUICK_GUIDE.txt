6. Create commit:
   git commit -m "Initial commit: Ticket Agent"

7. Add remote:
   git remote add origin https://github.com/DHARSAN777/AI-Ticket-Routing-Agent.git

8. Push to GitHub:
   git push -u origin main

9. If prompted for password: Use Personal Access Token (not password)
   - Generate token: https://github.com/settings/tokens
   - Create token with "repo" scope
   - Paste when prompted


🔐 AUTHENTICATION (Important!)
═══════════════════════════════════════════════════════════════════════════

Option 1: Personal Access Token (Recommended)
  1. Go to: https://github.com/settings/tokens
  2. Click: "Generate new token"
  3. Name: "Git Push"
  4. Scope: Select "repo" (full control)
  5. Expiration: 90 days
  6. Generate and COPY the token
  7. Use as password when git prompts

Option 2: GitHub CLI
  1. Download: https://cli.github.com/
  2. Run: gh auth login
  3. Follow prompts
  4. Automatically authenticated

🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

ERROR: "Git not found"
  → Install Git from https://git-scm.com/download/win

ERROR: "Authentication failed"
  → Use Personal Access Token (not password)
  → Generate at https://github.com/settings/tokens

ERROR: "Remote already exists"
  → Run: git remote remove origin
  → Then: git remote add origin <url>

ERROR: "Nothing to commit"
  → Run: git add .
  → Check status: git status

ERROR: "Please configure user"
  → Run: git config --global user.name "Your Name"
  → Run: git config --global user.email "your@email.com"

ERROR: "fatal: not a git repository"
  → Make sure you're in project folder
  → Run: git init
