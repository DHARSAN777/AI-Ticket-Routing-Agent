## 🔐 When Prompted for Password

**Username:** Your GitHub username  
**Password:** Use your Personal Access Token (NOT your password)

### How to Get Personal Access Token:

1. Go to: https://github.com/settings/tokens
2. Click: "Generate new token"
3. Fill in:
   - **Name:** "Git Push"
   - **Expiration:** 90 days
   - **Scope:** Check ✓ "repo" (full control)
4. Click: "Generate token"
5. **COPY** the token (you won't see it again!)
6. **PASTE** when git asks for password

## 🚨 If Something Goes Wrong

### Error: "fatal: not a git repository"
**Solution:** Make sure you ran `git init` first

### Error: "fatal: remote origin already exists"
**Solution:** Run first: `git remote remove origin`

### Error: "Authentication failed"
**Solution:** Use Personal Access Token, not password
- Generate new token: https://github.com/settings/tokens
- Use token as password

### Error: "nothing to commit"
**Solution:** Run: `git add .` first

### Error: "Could not resolve host"
**Solution:** Check internet connection
