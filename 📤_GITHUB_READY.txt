🔐 GET PERSONAL ACCESS TOKEN
═══════════════════════════════════════════════════════════════════════════

1. Go to: https://github.com/settings/tokens
2. Click: "Generate new token"
3. Fill in:
   - Name: "Git Push"
   - Expiration: 90 days
   - Scope: Check "repo"
4. Click: "Generate token"
5. COPY the token
6. PASTE when git asks for password

🆘 IF SOMETHING FAILS
═══════════════════════════════════════════════════════════════════════════

Error: "git: command not found"
  → Install Git from https://git-scm.com/download/win
  → Restart Command Prompt after install

Error: "Authentication failed"
  → Use Personal Access Token (not your password)
  → Generate new token: https://github.com/settings/tokens

Error: "fatal: not a git repository"
  → Make sure you ran: git init

Error: "fatal: remote origin already exists"
  → First run: git remote remove origin
  → Then run: git remote add origin ...

Error: "nothing to commit"
  → Run: git status (should show files)
