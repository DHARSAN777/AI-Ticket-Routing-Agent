## ⚡ Quick Start (All Platforms)

### Windows (Fastest)
```bash
# Just run this:
START-HERE.bat
```

### Mac/Linux
```bash
# Terminal 1: Backend
cd backend
python3 -m pip install -r requirements.txt
python3 -m uvicorn main_simple:app --host 0.0.0.0 --port 5000

# Terminal 2: Frontend
cd frontend
python3 -m http.server 8000
```

### Docker
```bash
docker build -t ticket-agent .
docker run -p 5000:5000 -p 8000:8000 ticket-agent
```

## 🎯 Common Questions

### Q: Do I need a database?
**A:** No, it works completely offline with in-memory storage. Optional: Connect MongoDB for persistence.

### Q: Do I need API keys?
**A:** No! The AI works with keyword matching, no external APIs needed.

### Q: How long does setup take?
**A:** 5 minutes with `START-HERE.bat`, 30 minutes with manual setup.

### Q: Can I customize it?
**A:** Absolutely! Full source code included. Change colors, categories, logic, anything.

### Q: Can I deploy to production?

### Q: Can I integrate with Slack/email?
